Sobre el Proyecto

A Aclimate Angola é uma plataforma agroclimática desenvolvida no contexto de um projeto consultivo voltado para a criação de uma plataforma de apoio à tomada de decisões no setor financeiro agrícola em Angola. Esse esforço é parte integrante do Projeto de Desenvolvimento da Agricultura Comercial (PDAC), com o objetivo de aprimorar a produtividade e facilitar o acesso ao mercado para beneficiários selecionados nas províncias de Malanje, Cuanza Norte, Cuanza Sul, Huambo, Bié e Fuja.

Essa plataforma agroclimática fornece informações específicas para áreas de produção de milho, feijão, soja e café nas províncias mencionadas. Ela é composta por três módulos fundamentais:

  1. Visualizador de Mapas Agroclimáticos: Este módulo possibilita que os usuários, em sua maioria extensionistas, equipes técnicas e órgãos decisórios, visualizem dados históricos espaciais relacionados a indicadores agroclimáticos. Esses indicadores abrangem necessidades hídricas das plantações, períodos secos, períodos chuvosos, ondas de calor, déficits hídricos e excesso de água. Através desse módulo, os usuários têm a oportunidade de explorar como diferentes fases do fenômeno El Niño-Oscilação Sul (ENSO), como "El Niño" e "La Niña", impactaram historicamente as demandas hídricas nas regiões produtoras de milho, trigo, soja e café. Além disso, é possível identificar áreas agrícolas mais suscetíveis a secas, inundações e ondas de calor.
  2. Previsões Climáticas Sazonais: Este módulo fornece informações sobre previsões climáticas sazonais, tanto probabilísticas quanto determinísticas, para precipitação, temperaturas e radiação solar em nível municipal. Essas projeções são atualizadas mensalmente, com um horizonte de até seis meses e agrupadas em trimestres. Além disso, os usuários têm acesso a dados históricos mensais (em forma de gráficos e tabelas) das variáveis climáticas mencionadas, referentes às localidades de interesse.
  3. Previsão de Produtividade: Este módulo possibilita o acesso a informações sobre a previsão de produtividade das culturas de milho, feijão e soja, considerando as condições climáticas projetadas para os seis meses subsequentes. Aqui, os usuários podem gerar projeções para o desempenho das culturas mencionadas, levando em conta diferentes épocas de plantio, tipos de solo e variedades específicas da região de estudo. A informação disponibilizada por este módulo permite que os usuários identifiquem as melhores datas de plantio para otimizar o rendimento das culturas.
